# Dr. Anupom Das – Academic Portfolio

This is the personal academic portfolio website of **Dr. Anupom Das**.

## Features
- Academic / PhD-ready professional website
- English / Bengali language toggle
- Website search
- Publications and research profile
- Projects and technical skills
- CV download
- Mobile responsive design

## Files
- `index.html`
- `style.css`
- `script.js`
- `photo.jpg`
- `CV.pdf`

## To Use
1. Upload all files to GitHub repository
2. Add your photo as `photo.jpg`
3. Add your CV as `CV.pdf`
4. Enable GitHub Pages

## GitHub Pages Setup
- Go to **Settings**
- Click **Pages**
- Select **Deploy from branch**
- Choose **main**
- Choose **/root**
- Save

Your site will be live after publishing.
